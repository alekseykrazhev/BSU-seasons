Вопрос 1 (бэш):
BASH, расшифровка - Born Again Shell, командный интерпретатор, с помощью которого пользоваьель может работать в интерактивном режиме (с использованием консоли или терминала).
У бэша есть собственный язык команд. Он состоит из переменных и операций над ними, циклов и тд
Переменные - именнованое значение, принимает числовые, символьные и строковые значения, массивы. Бывают глобальные (например env, printenv) и локальные (set)
var="value" - присвоение локальной переменной
export var="value" - глобальной
Также можно экспортировать и локальные переменные (экспорт в среду окружения) export local_var
При создании переменной нельзя называть ее с цифры, без пробелов, имена регистрозависимы
Ссылка на значение переменной - ${var}
Операции:
определить колво символов - ${#var}
${var:-word} - изменение значения
${var:+word} - изменение значения существующей переменной
#{var:=word} - присвоение значения переменной
Удаление переменной - unset [-v] var (можно использовать дополнительные опции к этой команде)
Создание массива: arr[i]="val", arr=(var1 var2) 
Ссылка на значение элемента массива: ${var[ind]}
Удаление массива: unset arr
Удаление элемента: unset arr[i]
Псевдонимы или синонимы  —  это краткие обозначения команд. Создаются командой alias name="comand"
Выход из оболочки - exit
Сценарий бэш - текстовый файл, содержащий команды (инструкции) командной оболочки. Исполняется интерпретатором.
Правила: 1 строка - 1 команда, команды разделяются ;, перенос команды: \, коментарии #, первая строка:
#! bin/bash
Запускаются из командной оболочки: ./filename.sh

Вопрос 2:
История, Архитектура Андроид.
Андроид - самая популярная мобильная операционная система. Основана на ядре Линукс. Первая стабильная версия вышла в 2008г.
Изначально андроид разрабатывала компания Android, которую позже приобрела Гугл (в 2005г.). На данный момент есть 12 версий андроида, разрабатывается 13. Названия содержат названия сладостей, и первые буквы идут в порядке латинского алфавита. Первая неофициальная версия вышла в 2007г. (в виде эмулятора).
В версии 4,0 добавилась поддержка планшетов, 5.0 - смена дизайна. 2.2 - режим модема, 2.3 - улучшение производитнльности + поддержка нескольких камер. 3.0 - ориентирована на планшеты.
В системе ядро линукса отвечает за управление памятью, поддержку мультизадачности, мультиплатформенности и за безопасность данных. Ядро линукса в андроид модифицировано.
Вместо libc в андроид используется минималистичная версия bionic (она оптимизирована для встраиваемых систем).
Вместо Busybox используется Toybox.
В андроид есть оболочка командной строки + много команд(программ)
Эмулятором терминала может быть Termux или похожие приложения.
Также в андроид используется FreeType, SQLite и графичиские апишники
Части архитектуры андроид:
1. Системные (application)
calendar, home, phone, ...
2. Java API(Application Framework)
view system, content provider, ...
3. Библиотеки С и С++
SQLite, SSL, media framework, ...
4. Android runtime
библиотеки ядра, ...
5. Hardware Abstraction layer
camera, audio, ...
6. ядро Linux
power management, drivers, ...
